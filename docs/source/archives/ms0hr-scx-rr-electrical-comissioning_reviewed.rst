First time switching on the power supply to the MS
==================================================

Before first electrical power on, all connections between MS and control
cabinet should be connected according specific electrical circuit
diagram for specific project and checked.

When MS control voltage +24VDC (Auxiliary supply) and motor power supply
MS starts operating according algorithm presented on figure 16.
Algorithm present general view of the sequence of MS operation states.
Likewise the algorithm executes after MS reset.

MS operational states are indicated on 7 – segment display of MS
controller and on MSHMI\*.

*\*MSHMI is not part of standard MS equipment.*

|image35|

**Figure 16** General view of the sequence of MS operation states

First operation is MS controller internal check. Internal check is
intended to hardware and firmware healthy of MS controller, MS motor and
MS motor resolver. If no Recovery mode is request after MS controller
internal check finishes, Self-test operation mode (further called self –
test) start. After successfully passed self – test MS is in Following
operation state. This operation state allows MS to follow hoist movement
and respectively normal hoist operation.

Self-test
---------

Self – test operation state is very important part of MS operating. The
test check electrical and mechanical components of MS. All states of
self – test are presented on MS controller 7 – segment display (figure
17). Figure 18 present self-test steps.

+-----------+------------------+-----------+--------------------------+

Following
---------

General block diagram of following operation state of MS is presented on
figure 19. MS is waiting for movement request from hoist. After request
is received, MS is waiting for movement of hoist (hoist barrel). If
hoist start moving, MS starts following.

During following, MS is monitoring for overspeed. In case of overspeed
detection, MS is tripped and prohibits hoist movement. Message for
overspeed is displayed on MS controller 7-segment display. MS is waiting
for restart.

If no overspeed is detected, MS is following until request form move is
active. In case there is no request for movement, but movement is still
active, MS is following hoist until it stops. When MS detect zero speed,

Figure 19 is presents main principle of *Following operation mode* and
overspeed detection.

|image57|

+-------------------------+--------------------------------------------+
| 1 – Acceleration        | 4 – Exceeding nominal speed                |
+=========================+============================================+
| 2 – Following           | 5 – Overspeed detection                    |
+-------------------------+--------------------------------------------+
| 3 – Deceleration        | 6 – Overspeed detected. MS actuated        |
+-------------------------+--------------------------------------------+
